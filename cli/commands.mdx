Binary: `fare`

## Overview

- init: Initialize and deploy a new casino. See `/cli/commands/init`.
- deploy: Rebuild and redeploy from the current casino directory. See `/cli/commands/deploy`.
- status: View deployment health and URLs. See `/cli/commands/status`.
- logs: Stream logs (api, ws, processor). See `/cli/commands/logs`.
- cleanup: Destroy local and Fly resources (dev-only). See `/cli/commands/cleanup`.
- keygen: Generate Solana keypairs for config. See `/cli/commands/keygen`.
